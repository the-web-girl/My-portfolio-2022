Cours d'HTML pour les débutants

Le HTML (Hypertext Markup Language) est un langage de balisage utilisé pour créer des pages web. Il permet de structurer le contenu d'une page en utilisant des balises pour définir différents éléments tels que les titres, les paragraphes, les images, etc. Dans ce cours, nous allons apprendre les bases du HTML, de sa création à nos jours.

Histoire du HTML

Le HTML a été développé par Tim Berners-Lee en 1990 au CERN (Organisation Européenne pour la Recherche Nucléaire) en Suisse. Le premier navigateur web, appelé WorldWideWeb, a été créé en 1991, permettant aux utilisateurs d'accéder à des pages web. La première spécification officielle du HTML a été publiée en 1995 par le W3C (World Wide Web Consortium), l'organisme de standardisation du web.

Structure de base du HTML

La structure de base du HTML comprend les éléments suivants :

  • La balise !DOCTYPE qui indique la version du HTML utilisée
  • La balise html qui entoure tout le contenu de la page
  • La balise head qui contient les informations sur la page tels que le titre, les méta-données, etc.
  • La balise body qui contient le contenu visible de la page

La structure de base du HTML est importante car elle définit l'organisation et la hiérarchie du contenu de la page. Elle permet également aux navigateurs web de comprendre comment afficher la page. La balise !DOCTYPE indique la version du HTML utilisée, ce qui est important pour la compatibilité avec les navigateurs. La balise html entoure tout le contenu de la page et la balise head contient les informations sur la page, telles que le titre, les méta-données, etc. La balise body contient le contenu visible de la page, tel que les titres, les paragraphes, les images, etc.

Les balises HTML

Les balises HTML permettent de définir les différents éléments de la page, ces balises HTML valides jouent un rôle important dans l'accessibilité numérique. Voici une liste non exhaustive de balises couramment utilisées :

  • h1 à h6 : utilisées pour les titres de différents niveaux, elles permettent aux utilisateurs de comprendre la structure de la page et la hiérarchie du contenu.
  • p : utilisée pour les paragraphes, elle permet de structurer le texte en blocs, facilitant ainsi la lecture.
  • img : utilisée pour les images, elle permet d'ajouter des descriptions alternatives (alt) pour les utilisateurs non-voyants ou malvoyants.
  • a : utilisée pour les liens, elle permet d'ajouter des textes descriptifs (title) et de créer des liens internes ou externes.
  • ul et li : utilisées pour les listes non-ordonnées, elles permettent de structurer l'information de manière logique et cohérente.
  • ol et li : utilisées pour les listes ordonnées, elles permettent d'indiquer clairement la séquence ou l'ordre des éléments.
  • input : utilisée pour les champs de saisie, elle permet d'ajouter des étiquettes (label) pour décrire le champ et son utilisation.
  • button : utilisée pour les boutons, elle permet d'ajouter des textes descriptifs (title) pour décrire l'action à effectuer.
  • table : utilisée pour créer des tableaux, elle permet de structurer l'information de manière lisible et cohérente.
  • form : utilisée pour créer des formulaires, elle permet aux utilisateurs de saisir des informations et de les envoyer au serveur.
  • label : utilisée pour étiqueter les champs de saisie, elle permet aux utilisateurs de comprendre ce qui est attendu d'eux.

En utilisant ces balises de manière appropriée, les développeurs peuvent aider les utilisateurs à naviguer sur les pages plus facilement et à comprendre la structure de l'information. Les descriptions alternatives (alt) pour les images et les textes descriptifs (title) pour les liens permettent également aux utilisateurs non-voyants ou malvoyants de comprendre le contenu de la page. Enfin, l'utilisation de balises sémantiques (comme header, footer, nav, article...) permettent de décrire la structure de la page et les différents blocs de contenu, facilitant ainsi l'accessibilité et l'interprétation du contenu par les utilisateurs.

Les attributs HTML

Les attributs HTML permettent de donner des informations supplémentaires aux balises. Voici quelques exemples d'attributs couramment utilisés :

  • src pour spécifier l'emplacement de l'image à afficher
  • alt pour fournir une description textuelle de l'image
  • href pour spécifier l'URL de la page liée
  • class pour attribuer une classe CSS à un élément
  • id pour donner un identifiant unique à un élément

Les éléments HTML en ligne et en bloc

Les éléments HTML peuvent être soit en ligne, soit en bloc. Les éléments en ligne sont affichés les uns à côté des autres, tandis que les éléments en bloc occupent toute la largeur de la page. Voici quelques exemples d'éléments en ligne et en bloc :

  • En ligne : a, span, img
  • En bloc : div, p, ul

Conclusion

Le HTML est un langage de balisage essentiel pour la création de pages web. Il permet de structurer le contenu de la page et de définir les différents éléments tels que les titres, les paragraphes, les images, etc. En utilisant des feuilles de style CSS, il est possible de personnaliser l'apparence de ces éléments pour créer des pages web attrayantes et fonctionnelles.
